Nessy the NES Robot is a homebrew NES game created by Daniel T. Gaming and was originally made for Byte-Off 2019. This game was made using a game development software created by The New 8-Bit Heroes named "NESmaker", which allows you to create your own NES games, distribute them, and play them on either an emulator or actual hardware.

The first timesave is in Level 1. In my 2-minute TAS, I shot one of the enemies. This lagged the game for one frame, resulting in me losing one frame.

To save that one frame, I jumped over the enemy instead of shooting it. This caused no lag, which made this TAS one frame faster in level 1.

Level 3 Boss Skip

Another timesave is in Level 3.

NESmaker (the game engine Nessy was made with) uses parts of a map named "screens", which contain the game's levels.

In Nessy, the screens scroll horizontally, so going right will scroll the screen next to the screen you're on to the left. However, if you go to the bottom of the screen you're on, you will be teleported to the screen below it.

Coincidentally, the exit screen for the boss fight is underneath one of the regular screens, so if you manage to get to the bottom of that screen, you can skip the boss fight.

In order to get to the bottom of that screen, I had to enter the door to that screen with enough falling speed to get underneath the ground and reach the bottom of the screen, skipping Level 3's boss fight entirely.
